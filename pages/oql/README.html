OQL — Omega Query Language

OQL is a logic programming language inspired by Datalog and Prolog. You describe relationships and constraints; the engine finds all consistent solutions. Every statement is a term that either succeeds or fails against the current solution set. Understanding this model is the foundation for everything else in this section.
